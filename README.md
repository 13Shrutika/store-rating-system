**Store Rating System**

A full-stack web application that allows users to register, browse stores, and submit ratings (1–5). The system supports role-based access control for Admin, 
Normal Users, and Store Owners.

**Tech Stack**

**Frontend**
- React.js
- Axios
- React Router DOM
- CSS 
**Backend**
- Node.js
- Express.js
- MySQL
- JWT Authentication
- bcrypt.js

**User Roles**

  
**1. Admin**
- Add users (Admin / User / Owner)
- Add stores
- View all users
- View all stores
- View dashboard stats

**2. Normal User**
- Register/Login
- View all stores
- Submit rating (1–5)
- Update rating

**3. Store Owner**
- View average rating of their store
- View users who rated their store

**Features**

1. JWT Authentication
2. Role-based access control
3. Store management system
4. Rating system (1–5)
5. Owner analytics dashboard
6. Search & filtering (admin side)
7. Update ratings
8. Logout functionality

**Installation Guide**
1. Clone Repo
2. **Backend Setup**
   - cd backend
   - Npm install
3. npm run dev 
4.**Frontend setup**
   - cd frontend
   - npm install
   - npm start
